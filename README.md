## Research Projects

### LWR-PINN for Traffic Density Reconstruction under Sparse Probe Data

- Studied LWR-based Physics-Informed Neural Networks for traffic density reconstruction under sparse probe data, showing improved structural reconstruction performance over pure MLP baselines.
- Compared PINN and MLP under 2%, 5%, and 10% probe ratios with spatial and temporal missing conditions.
- Evaluated reconstruction quality using RMSE, SSIM, and Active RMSE.

### PITAM: Physics-Informed Active Sensing for Traffic Density Field Reconstruction

- Proposed PITAM, a physics-informed active sensing framework that reconstructs traffic density fields while reducing cumulative uplink communication by 17.7%.
- Designed RSU and Probe selection strategies using PDE residual, density gradient, shock-wave proximity, and uncertainty.
- Modeled asymmetric missing scenarios where Probe communication is lost in tunnel segments while RSU sensing remains available.

### PIGNN-Event: Physics-Informed Gating Neural Network for Traffic Congestion Prediction

- Developed PIGNN-Event, a physics-informed gated STGCN model for 3-class traffic congestion prediction in heterogeneous RSU-Probe sensing environments.
- Converted LWR-PINN reconstructed density fields into congestion event priors.
- Combined physics priors with STGCN-ResNorm representations through a gating module for 5-minute-ahead link-level congestion prediction.
